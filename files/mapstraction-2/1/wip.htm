Mapstraction v2

No working demo yet, this is very much a work in progress. My aim is to get a core set of functionality working in Google with the key features being:

  • disconnected provider implementation
  • dynamic creation of basic proxy methods which pass straight though to underlying implementation
  • invoking helper class for calling the correct provider implemention from the mapstraction classes
  • improved events allowing you to handle a lot more events on both the map and markers

mxn.js
Base library with helper methods and classes
mxn.core.js
Core Mapstraction classes (Mapstraction, LatLonPoint, BoundingBox, Marker, Polyline)
mxn.google.core.js
Google implementation of core classes
